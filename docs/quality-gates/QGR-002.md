# 🧪 Estratégia de Testes

## Tipo de teste escolhido

- [X] Smoke Test
- [ ] Regression Test
- [ ] Sanity Test

---

## Justificativa

O Smoke Test foi escolhido pois faz validações básicas em funcionalidades críticas no sistema.

Não utilizamos o Regression Test pois ele necessita de uma validação mais aprofundada em funcionalidades que se comunicam com outras funcionalidades dependentes do sistema.
Com relação ao Sanity Test, não é adequado pois ele cobre apenas mudanças pontuais, como um hotfix.
